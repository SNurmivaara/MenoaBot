# A simple ruby telegram bot for returning usage stata of a certain space from an API

## Usage

1. Set token and api to use
2. Run ruby bundle install
3. Create heroku app
4. Push to heroku
5. Activate dyno
6. Enjoy!

## Notes

You might have to transform the API calls to match the API used.
